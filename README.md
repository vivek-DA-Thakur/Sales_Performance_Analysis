## Project: Sales Performance Analysis

### Objective
Analyze sales data to identify regional, category-wise, and seasonal trends.

### Dataset
- Source: Kaggle (Superstore Sales Dataset)
- Rows: ~10k
- Columns: Orders, Sales, Profit, Region, Category, Date

### Tools Used
- SQL (SQLite)
- Python (Pandas, Matplotlib)

### Key Questions Answered
- Which regions are most profitable?
- Which categories lose money?
- How does profit change over time?

### Key Insights
- West region has highest profit margin (~15%)
- Furniture (Tables) consistently loses money
- Q4 shows strong seasonal peaks

### Business Recommendations
- Optimize pricing for Furniture
- Prepare inventory for Q4 demand
- Track profit margins alongside sales
